API de práctica de un CRUD simple

  • https://crud-flask-clases.vercel.app/api/Buscar/{id} - Obtiene la información de un documento en específico
  • https://crud-flask-clases.vercel.app/api/Todo - Obtiene todos los campos
  • https://crud-flask-clases.vercel.app/api/Eliminar/{id} - Elimina un documento específico mediante un ID proporcionado
  • https://crud-flask-clases.vercel.app/api/Agregar - Agrega un documento a la BDD
